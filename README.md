# Excel-Finance-Analytics-AtliQ-Hardware
Finance Analytics Dashboard built in Microsoft Excel for AtliQ Hardware using Power Query, Pivot Tables, Power Pivot, DAX, and interactive dashboards to analyze sales performance and financial insights.

📊 Finance Analytics Dashboard – AtliQ Hardware
📌 Project Overview

This project is part of the Codebasics Excel Course, where I developed an interactive Finance Analytics Dashboard for AtliQ Hardware using Microsoft Excel.

The dashboard enables users to analyze financial performance, monitor net sales trends, compare yearly growth, and evaluate customer performance through dynamic reports and visualizations.

🎯 Objective

The objective of this project is to transform raw sales data into meaningful financial insights that help business stakeholders make informed decisions.

From the dashboard, users can quickly identify top-performing customers, compare yearly sales, and evaluate growth trends across different markets.

📊 Key Insights
Net sales increased significantly from 2019 to 2021.
2021 recorded the highest overall sales performance.
Amazon, AtliQ Exclusive, and AtliQ e Store were among the top-performing customers based on net sales.
The dashboard supports dynamic filtering by customer, region, market, and division, making financial analysis more flexible.
